# 🧠 MGBX 智能量化

> **4 小时判大势，15 分钟抓拐点。50x 杠杆，保证金驱动仓位。**
>
> 下跌趋势只找"涨不动"做空。上涨趋势只找"跌不动"做多。
> **不逆势，不扛单，不猜底。**

## ⚖️ 铁律（不可违反）

| # | 规则 | 说明 |
|---|------|------|
| 1 | **不逆大势** | 4H 下跌只做空，上涨只做多，NEUTRAL 双向 |
| 2 | **三次熔断** | 单日止损达 3 次 → 强制休息至次日 |
| 3 | **止损必绑** | 每笔开仓同步设止损，无例外 |

| 低点连续抬高 + 后半均价微涨 | **UP ↑** | 中 |
| 以上均不满足 | **NEUTRAL** | — |

> 宏观方向每完成一根 4H K 线刷新一次。

```bash
python3 ~/.mgbx/mgbx_api.py balance CONTRACT
```

### ⚡ 小本金仓位公式

```
每张保证金 = 当前价格 × 0.0001（合约面值）÷ 实际杠杆

实际杠杆 = 50 × 60% = 30x
可用保证金 = 净值 × 60%

可开张数 = ⌊ 可用保证金 ÷ 每张保证金 ⌋（最少 1 张）
```

| 净值 | 最大杠杆 | 实际杠杆 | 保证金比例 |
|------|---------|---------|-----------|
| < $10K | **50x** | 30x | 60% |
| $10K–$100K | 20x | 12x | 40% |
| $100K–$1M | 10x | 6x | 25% |
| > $1M | 5x | 3x | 15% |

> 💡 小本金的核心优势：50x 下每张保证金极低（$0.15–$0.25），$20 就能开几十张。

## 📋 Step 4：15 分钟开仓信号

```bash
python3 ~/.mgbx/mgbx_api.py kline btc_usdt 15m 100
```

### 🟢 做多信号「跌不动了」

**前置**：4H 必须是 **UP** 或 **NEUTRAL**

**触发**：最近 4 根 15min K 线，至少 3 根低点逐步抬高

```
L₁ < L₂ < L₃  ← 确认跌不动
```

| 参数 | 计算 |
|------|------|
| 入场 | 当前收盘价（市价） |
| 止损 | 近 20 根最低点 × 0.99 |
| 止盈 | 入场 + (入场 − 止损) × 2.5 |
| 过滤 | 止损距离 > 5% 则不参与 |

### 🔴 做空信号「涨不起来了」

**前置**：4H 必须是 **DOWN** 或 **NEUTRAL**

**触发**：最近 4 根 15min K 线，至少 3 根高点逐步降低

```
H₁ > H₂ > H₃  ← 确认涨不动
```

| 参数 | 计算 |
|------|------|
| 入场 | 当前收盘价（市价） |
| 止损 | 近 20 根最高点 × 1.01 |
| 止盈 | 入场 − (止损 − 入场) × 2.5 |
| 过滤 | 止损距离 > 5% 则不参与 |

## 📋 Step 5：持仓管理（最终策略定型）

### ⭐ 核心机制：1.5R 保本 → 2R 追踪止盈

```
浮盈 < 1.5R  →  初始止损不变
浮盈 ≥ 1.5R  →  🛡️ 止损移至成本价（保本）
浮盈 ≥ 2.0R  →  🚀 启动追踪止盈（步长 2%，只进不退）
浮盈 ≥ 2.5R  →  到达止盈价，平仓锁利
```

| 阶段 | 触发条件 | 动作 |
|------|---------|------|
| 🛡️ 保本止损 | 浮盈 ≥ 1.5R | 止损移至入场价，确保不亏 |
| 🚀 追踪止盈 | 浮盈 ≥ 2.0R | 止损向有利方向移动 2%，价格每创新高/新低时更新 |
| 🎯 止盈 | 浮盈 ≥ 2.5R | 到达止盈价，全额平仓 |

### 📈 追踪止盈详解

```
做多追踪：
  价格每创新高 → 新止损 = 新高 × 0.98
  止损只升不降，锁定利润

做空追踪：
  价格每创新低 → 新止损 = 新低 × 1.02
  止损只降不升，锁定利润
```

### 🔼 浮盈加仓（可选）

| 条件 | 要求 |
|------|------|
| 浮盈率 | > 10%（已触发保本后） |
| 宏观方向 | 与持仓同向 |
| 累计加仓 | < 2 次 |
| 今日止损 | < 3 次 |

加仓量 ≤ 底仓量 50%，加仓部分独立止损（入场价 ± 3%）。

### 🛑 强制平仓

| 触发条件 | 动作 |
|----------|------|
| 触及止损价 | 立即全平，`daily_stops + 1` |
| 触及止盈价 | 平仓锁利 |
| 宏观方向反转 | 4H 方向与持仓相反 → 立即平仓 |

```bash
python3 ~/.mgbx/mgbx_api.py closeall btc_usdt
```
